## ❓ FAQ for Team

**Q: Why transformation messaging instead of feature-focused?**
A: Testing shows emotional narratives drive 3-5x higher conversion than feature lists. Week 6 test will validate.

**Q: Why 3 personas? Isn't that overcomplicating?**
A: 3 personas capture 95%+ of market. More than 3 = overkill. Different CTAs per persona drive +40-60% conversion improvement.

**Q: Why spend 2 weeks per A/B test?**
A: Sample size needs (1000+ visitors) + statistical rigor (95% confidence). Faster tests = unreliable results.

**Q: What if Test 1 winner doesn't increase conversion?**
A: Still ship it if non-negative. Test 2-4 compound learnings. Rarely is single test decisive.

**Q: Can we start implementation before strategy approval?**
A: No. Design needs final copy direction. Dev needs final designs. Testing needs approval. Sequential.

**Q: What if we don't have testimonials ready?**
A: Placeholder with generic quotes until real ones ready. Test 4 (Week 12) can still run.

**Q: Is feature flag complexity worth it?**
A: Yes. 2-3 hours dev time for instant rollback ability + baseline validation is insurance.

**Q: Can we launch faster than 5 weeks?**
A: Possible but risky (copy quality, design polish, testing coverage). 5 weeks is optimal sustainable pace.
