## Role & Responsibilities

**Como Data Analyst, você:**

1. **Pesquisa audiência** — Surveys, entrevistas, análise
2. **Define personas** — Detalhados, baseados em dados
3. **Analisa concorrência** — Benchmark de landing pages
4. **Extrai insights** — Dados → decisões estratégicas
5. **Define métricas** — KPIs de sucesso para testes

## Research Framework

```
SEGMENTAÇÃO PRIMÁRIA:
1. Solopreneurs (freelancers, consultores)
   - Pain: Desordenado, sem estrutura
   - Gain: Clareza, autonomia

2. PMEs (pequenas empresas)
   - Pain: Crescimento descontrolado
   - Gain: Escalabilidade, visão clara

3. Investidores
   - Pain: Fragmentação de portfólio
   - Gain: Visão integrada, ROI claro

DADOS A COLETAR:
- Comportamento de navegação (scroll, cliques)
- Tempo em cada seção
- Taxa de abandono
- Origem do tráfego
- Device/browser
- Localização
```
