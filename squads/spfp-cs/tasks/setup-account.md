## O que faz
- Ajuda o cliente a configurar a conta no SPFP (por chamada ou via materiais)
- Guia o cadastro das contas bancárias e cartões
- Explica como categorizar transações
- Envia materiais de apoio: vídeos tutoriais, guia de início rápido
- Confirma que o cliente conseguiu realizar a configuração básica
- Registra status da configuração no CRM

## Não faz
- Fazer a configuração pelo cliente (cliente precisa aprender)
- Criar a conta do cliente no sistema
- Resolver bugs técnicos durante o setup (encaminha para Suporte)

## Ferramentas
- Zoom (chamada de setup para clientes premium)
- Loom (vídeos tutoriais gravados)
- Notion (guia passo-a-passo)
- CRM (registro do progresso)

## Checklist de setup completo

- [ ] Conta criada e acesso confirmado
- [ ] Pelo menos 1 conta bancária cadastrada
- [ ] Primeiras categorias configuradas
- [ ] Cliente sabe como lançar uma transação
- [ ] Materiais de apoio enviados
- [ ] Perguntou se ficou alguma dúvida

## Materiais de apoio padrão

1. **Vídeo**: "Primeiros 5 minutos no SPFP" (Loom, < 3 min)
2. **Guia**: Checklist de configuração (Notion/PDF)
3. **FAQ**: As 10 dúvidas mais comuns no setup
