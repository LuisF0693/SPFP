# Workflow: Product Creation

## Entrada: Oportunidade identificada ou feedback de EXPERIÊNCIA

---

## PRODUCT MANAGER

```
Discovery → Roadmap → Spec
```

### QUALITY GATE 1: Spec aprovada?
- **NÃO** → Refina Spec (PM ajusta e reapresenta)
- **SIM** → Content Creator começa

---

## CONTENT CREATOR

```
Research → Create
```

### QUALITY GATE 2: Conteúdo pronto?
- **NÃO** → Continua criando
- **SIM** → QA Produto entra

---

## QA PRODUTO

```
Quality Check → Test
```

### QUALITY GATE 3: Aprovado?
- **NÃO** → Volta para Creator (Review)
  - Creator faz Review
  - Volta para Quality Check → Test
- **SIM** → Continua

---

## PUBLICAÇÃO E LANÇAMENTO

```
Publish (Content Creator)
  → Launch Coordination (Product Manager)
    → Avisa MARKETING e VENDAS: Produto Pronto para lançar
```

---

## PÓS-LANÇAMENTO: FEEDBACK LOOP

```
EXPERIÊNCIA (FEEDBACK)
  → QA PRODUTO (ORGANIZA)
    → PM (PRIORIZA)
      → MELHORIA
```

O feedback loop alimenta novo ciclo de Discovery → e o ciclo recomeça.

---

## Resumo do fluxo completo

```
Entrada (Oportunidade/Feedback)
  ↓
PM: Discovery → Roadmap → Spec
  ↓
[QG1] Spec OK? → NÃO: Refina → SIM: continua
  ↓
Creator: Research → Create
  ↓
[QG2] Pronto? → NÃO: Continua → SIM: continua
  ↓
QA: Quality Check → Test
  ↓
[QG3] Aprovado? → NÃO: Creator Review → QA de novo → SIM: continua
  ↓
Creator: Publish
  ↓
PM: Launch Coordination → Avisa MKT + Vendas
  ↓
[Pós-lançamento]
Experiência → QA Organiza → PM Prioriza → Melhoria
```
