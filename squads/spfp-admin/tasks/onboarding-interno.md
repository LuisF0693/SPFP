# Task: Onboarding Interno
**Agente:** RH/People
**Input:** Candidato aprovado (Head + CEO decidiram)
**Output:** Acessos + equipamento + docs trabalhistas + colaborador integrado

---

## Objetivo
Garantir que o novo colaborador tenha tudo que precisa para começar a trabalhar no primeiro dia.

## Processo (ver checklist completo em agents/rh-people.md)

```
Antes do primeiro dia (D-3):
  • Email criado, Slack adicionado, ferramentas configuradas
  • Docs trabalhistas enviados para assinatura

Dia 0 (primeiro dia):
  • Welcome formal + apresentação da empresa e cultura
  • Reunião com CEO (30 min) e Head do squad
  • Buddy designado
  • Tour pelas ferramentas

Semana 1–4:
  • Check-ins de acompanhamento
  • NPS de onboarding no Dia 30
```

## Critérios de Done
- [ ] Acessos funcionando antes do primeiro dia
- [ ] Documentação trabalhista assinada
- [ ] Apresentação de cultura realizada
- [ ] Buddy designado
- [ ] NPS de onboarding coletado no Dia 30
