# Task: Offboarding
**Agente:** RH/People
**Input:** Desligamento aprovado (Head + CEO)
**Output:** Rescisão processada + acessos revogados + entrevista de saída realizada

---

## Objetivo
Garantir desligamento digno, completo e seguro — rescisão correta, acessos revogados no mesmo dia e aprendizado registrado.

## Processo

```
1. Desligamento aprovado por Head + CEO
2. RH coordena:
   D-1: Prepara rescisão e documentação
   D0 (dia do desligamento):
     • Revoga todos os acessos (Facilities executa imediatamente)
     • Processa rescisão trabalhista
     • Entrevista de saída (RH conduz)
   D+5: Rescisão assinada e homologada
   D+7: Relatório de entrevista de saída para Head Admin + CEO
```

## Entrevista de Saída — Perguntas-chave

1. Por que você está saindo? / O que te fez aceitar outra oportunidade?
2. O que o SPFP poderia ter feito diferente?
3. O que você mais gostou de trabalhar aqui?
4. Você recomendaria o SPFP como lugar para trabalhar? Por quê?
5. Há algo que você queira deixar como feedback para a liderança?

## Critérios de Done
- [ ] Acessos revogados no mesmo dia do desligamento
- [ ] Documentação trabalhista processada e assinada
- [ ] Entrevista de saída realizada
- [ ] Relatório de saída enviado ao Head Admin
- [ ] Conhecimento crítico do colaborador documentado antes de sair
