# Task: Recrutamento
**Agente:** RH/People
**Input:** Demanda de vaga (de qualquer squad)
**Output:** Candidatos triados + entrevistas agendadas

---

## Objetivo
Publicar vaga, atrair candidatos qualificados e entregar shortlist para o Head do squad contratar.

## Processo

```
1. Recebe demanda de vaga do squad (Job Description + requisitos + budget aprovado)
2. Valida: Head do squad aprovou a abertura? Budget confirmado?
3. Publica vaga (Gupy + LinkedIn)
4. Triagem de currículos (fit cultural + requisitos básicos — feita por RH)
5. Entrevista 1 (RH): cultura, motivação, comunicação, fit
6. Aprovados passam para entrevista técnica (squad responsável faz)
7. Shortlist de 2–3 finalistas para Head + CEO decidirem
8. Proposta formal ao aprovado
9. Processo de onboarding-interno inicia
```

## Critérios de Done
- [ ] Vaga publicada nos canais corretos
- [ ] Shortlist de 2–3 finalistas entregue ao Head do squad
- [ ] Entrevistas de RH realizadas e documentadas
- [ ] Candidatos não aprovados notificados com feedback
