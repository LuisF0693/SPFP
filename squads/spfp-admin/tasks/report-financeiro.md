# Task: Report Financeiro
**Agente:** Financeiro
**Input:** Dados financeiros do período (Conta Azul / Omie / Sheets)
**Output:** DRE mensal + relatório pro CEO (burn rate, runway, saúde financeira)

---

## Objetivo
Produzir relatório financeiro completo todo início de mês para o CEO e Head Admin.

## Frequência
- **Mensal:** Até o dia 5 do mês seguinte
- **Semanal:** Resumo de fluxo de caixa para Head Admin

## Template de Output
Ver `agents/financeiro.md` — seção "Template DRE Mensal"

## Critérios de Done
- [ ] DRE completo com todas as categorias
- [ ] Burn rate calculado
- [ ] Runway calculado (em meses)
- [ ] Comparativo com mês anterior
- [ ] Alertas de risco incluídos (se houver)
- [ ] Relatório entregue ao CEO e Head Admin até dia 5
