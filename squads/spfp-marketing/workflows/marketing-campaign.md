# Workflow: Marketing Campaign

## Entrada
Objetivo de campanha definido pelo Head de Marketing

---

## Fase 1: RESEARCH (Research Analyst)

```
Competitor Analysis → Trend Hunting → Swipe File
```

### QUALITY GATE 1: Tem Insights?
- **NÃO** → Pesquise mais (volta para Research)
- **SIM** → Continua para COPY

---

## Fase 2: COPY

```
Pede pro COPY:
  - Briefing com insights do Research
  - COPY entrega: textos, scripts, artes
```

### QUALITY GATE 2: Material Pronto?
- **NÃO** → COPY ajusta
- **SIM** → Distribui para os canais

---

## Fase 3: EXECUÇÃO PARALELA

### Social Media (Social Media Manager)
```
Content Planning → Publish → Engage
```

### Tráfego Pago (Media Buyer)
```
Campaign Setup → Optimize → Scale
→ Automation: Pede pro OPS
```

### Email (Email Strategist)
```
List Segment → Campaign Setup → Automation
```

---

## Fase 4: CONTEÚDO (Content Manager)

```
SEO Research → Content Planning → Publish
```

### QUALITY GATE 3: Tudo publicado?
- **NÃO** → Verifica Canal
- **SIM** → Continua

---

## Fase 5: REPORTS (Todos os canais)

Cada agente entrega seu relatório de canal:
- Social Media Manager → engajamento, alcance
- Media Buyer → ROAS, CPA, CPC
- Email Strategist → open rate, CTR, conversão
- Content Manager → tráfego orgânico, rankings

---

### QUALITY GATE 4: Meta atingida?
- **NÃO** → Otimiza ou para campanha
- **SIM** → ✅ Campanha SUCESSO
