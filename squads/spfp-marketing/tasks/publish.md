## O que faz
- Recebe artigo ou roteiro de vídeo finalizado e aprovado
- Publica artigo no WordPress com formatação correta
- Otimiza SEO on-page: title tag, meta description, headers H1/H2/H3, alt text de imagens
- Adiciona links internos para outros artigos do SPFP
- Adiciona links externos de autoridade (fontes confiáveis)
- Configura URL amigável e slug SEO-friendly
- Publica vídeo no YouTube com título, descrição e tags otimizadas
- Adiciona capítulos no YouTube (timestamps)
- Confirma publicação e indexação no Google Search Console

## Não faz
- Reescrever o artigo (devolve para COPY se precisar de ajustes)
- Publicar sem aprovação do Head de Marketing
- Alterar o título original sem consultar COPY e Head

## Ferramentas
- WordPress / Webflow (artigos)
- YouTube Studio (vídeos)
- Google Search Console (indexação)
- Yoast SEO / RankMath (plugin WordPress)

## Checklist SEO pré-publicação

**Artigo:**
- [ ] Title tag contém keyword principal (máx 60 caracteres)
- [ ] Meta description persuasiva com keyword (máx 155 caracteres)
- [ ] H1 único com keyword principal
- [ ] H2s e H3s estruturam o conteúdo logicamente
- [ ] Imagens com alt text descritivo
- [ ] Mínimo 3 links internos para outros artigos
- [ ] URL limpa e curta com keyword

**Vídeo YouTube:**
- [ ] Título contém keyword (máx 60 caracteres)
- [ ] Descrição com keyword no primeiro parágrafo
- [ ] Tags relevantes (10-15 tags)
- [ ] Thumbnail personalizada
- [ ] Capítulos com timestamps
