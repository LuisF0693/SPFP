# Tech Stack — Squad Branding SPFP

## Ferramentas de Design

| Ferramenta | Uso |
|-----------|-----|
| Figma | Design de identidade visual, Brand Guidelines, UI mockups |
| Canva Pro | Templates de redes sociais, materiais de marketing |
| Adobe Illustrator | Logo vetorial, assets de alta qualidade |
| Adobe Photoshop | Edição de imagens, mockups fotográficos |

## Ferramentas de Cor e Tipografia

| Ferramenta | Uso |
|-----------|-----|
| Coolors | Exploração e geração de paletas |
| Adobe Color | Verificação de harmonia e acessibilidade |
| Google Fonts | Biblioteca de fontes web |
| Font Pair | Combinações de fontes |

## Ferramentas de Pesquisa e Análise

| Ferramenta | Uso |

| App Store / Google Play | Análise de reviews |
| Social Mention | Monitoramento de menções |
| Notion | Documentação de Brand Guidelines |

| Ferramenta | Uso |

| Grammarly | Revisão ortográfica e gramatical |
