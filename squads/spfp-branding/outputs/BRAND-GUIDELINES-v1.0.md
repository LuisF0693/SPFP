## 1. IDENTIDADE ESTRATÉGICA

**Posicionamento:** O único planejador financeiro pessoal que pensa com você — não por você.

**Brand Ladder:**
```
ESPÍRITO:    Cada brasileiro realizando seus sonhos com consciência financeira
VALORES:     Clareza · Controle · Confiança · Ambição com método
BENEFÍCIOS:  Eliminar a angústia financeira / Tomar decisões com segurança
ATRIBUTOS:   Plataforma completa / Finn (AI advisor) / Interface clara / Seguro
```

**Arquitetura:** Marca única (Branded House) — SPFP separado da marca pessoal LF.

**Tagline aprovada:** "Seu dinheiro, finalmente com sentido."

---

## 2. FINN — O PERSONAGEM

**Nome:** Finn
**Papel:** AI Advisor + Parceiro (adaptativo por contexto)
**Inspiração:** Consultor financeiro pessoal que é amigo — inteligente, preciso, humano.

### Finn Advisor (modo análise)
- Ativa quando: dados preocupantes, análise mensal, projeções, alertas
- Tom: direto, baseado em dados, respeitoso com a inteligência do usuário

### Finn Parceiro (modo suporte)
- Ativa quando: meta atingida, onboarding, mês difícil, marcos de progresso
- Tom: caloroso, genuíno, encorajador sem ser falso

**Regra de ouro:** Finn nunca culpa. Finn nunca ignora. Finn nunca exagera.

## 3. SISTEMA VISUAL — DIREÇÃO "JOURNEY" (APROVADA)

### 3.1 Logo System

**Conceito:** Seta de traço único que forma implicitamente um "F" de Finn.
A seta curva levemente — sugere jornada, não só crescimento linear.

```
LOGO PRINCIPAL (horizontal)
────────────────────────────────────────
[símbolo ╱→]  SPFP
              ──────────────────
              Seu Planejador Financeiro Pessoal

Símbolo:   Traço único, peso 3px, terminação arredondada (round cap)
           Curva suave de 15° no ponto de inflexão
           Seta aponta para direita-cima (não reto pra cima)
           Cor: Blue-500 (#1E6FE8) ou Teal-500 (#00C2A0) — depende do fundo

Wordmark:  Plus Jakarta Sans SemiBold (600)
           Tracking: +40
           Cor: Navy-700 (#1B3A6B) — fundo claro
                Branco (#FFFFFF) — fundo escuro

Tagline:   Plus Jakarta Sans Regular (400)
           Tamanho: 40% do tamanho do wordmark
           Tracking: +80
           Cor: Gray-600 (#4B5563) — fundo claro
                Gray-300 (#D1D5DB) — fundo escuro
```

**Variações obrigatórias:**

| Variação | Uso |
|---------|-----|
| Principal (cor, horizontal) | Uso padrão em fundos brancos/claros |
| Invertida (branco, horizontal) | Fundos escuros, Navy, foto com overlay |
| Símbolo isolado | App icon, favicon, espaços mínimos |
| Vertical (símbolo + wordmark empilhados) | Quadrado, avatar de redes sociais |
| Monocromática (preto) | Documentos, bordado, gravação |

**Zona de proteção:** 2× a altura da letra "S" do wordmark em todos os lados.

**Tamanho mínimo:**
- Digital: 80px de largura (horizontal) / 40px (símbolo isolado)
- Impresso: 15mm de largura (horizontal) / 8mm (símbolo isolado)

**Usos incorretos:**
```
❌ Distorcer ou inclinar o logo
❌ Alterar proporções símbolo/wordmark
❌ Usar sobre foto sem overlay escuro (mínimo 40% opacidade)
❌ Recolorir fora da paleta oficial
❌ Adicionar sombra ou efeito ao logo
❌ Combinar com marca pessoal LF em qualquer material
❌ Usar abaixo do tamanho mínimo
```

#### Verificação WCAG AA (contraste mínimo 4.5:1 para texto)
```
Branco (#FFF) sobre Navy-700 (#1B3A6B)   → 9.4:1  ✅ AAA
Branco (#FFF) sobre Blue-500 (#1E6FE8)   → 4.7:1  ✅ AA
Gray-900 sobre White                      → 16.1:1 ✅ AAA
Gray-900 sobre Blue-100                   → 11.2:1 ✅ AAA
Navy-900 sobre Teal-500                   → 6.1:1  ✅ AA
Branco sobre Teal-500                     → 2.9:1  ❌ Não usar para texto
```

> **Regra Teal:** Apenas para ícones, backgrounds, bordas e elementos decorativos.
> Nunca para texto abaixo de 24px bold sobre fundo claro.

### 3.4 Avatar do Finn

#### Finn Advisor
```
Forma:        Círculo perfeito
Borda:        2px sólido, Blue-500 (#1E6FE8)
Fundo:        Gradiente radial Navy-900 → Blue-700
Expressão:    Olhos: dois círculos sólidos, branco, 6px
              Sobrancelhas: duas linhas de 8px, espessura 1.5px
              Inclinadas levemente (-5° esquerda, +5° direita) — foco/concentração
              Boca: linha neutra horizontal 10px
Detalhe:      Pequeno ícone de linha-de-gráfico (3px, white 60%) no canto inferior-direito
Tamanho base: 40×40px no app / 200×200px para materiais
Animação:     Pulso suave (scale 1.0 → 1.03 → 1.0) quando "processando" — 1.5s ease
```

#### Finn Parceiro
```
Forma:        Mesmo círculo
Borda:        2px sólido, Teal-500 (#00C2A0)
Fundo:        Gradiente radial Navy-700 → Teal-600
Expressão:    Olhos: mesmos dois círculos
              Sobrancelhas: retas e levemente elevadas — abertura/acolhimento
              Boca: curva suave pra cima (arco de 12px, 3° de curvatura)
Detalhe:      Pequena estrela ou check (3px, Teal-200) no canto inferior-direito
Animação:     Bounce suave (translateY 0 → -3px → 0) ao aparecer — 0.4s ease-out
```

#### App Icon (1024×1024px)
```
Fundo:        Navy-900 (#0A1628) — sólido, sem gradiente
Elemento:     Letra "F" em Plus Jakarta Sans ExtraBold
              Cor: Blue-300 (#6AA9F4)
              Tamanho: 52% do ícone (532px)
              Centralizado horizontal e verticalmente (+2% pra cima opticamente)
Detalhe:      Ponto circular Teal-500 (#00C2A0) — 80px
              Posicionado sobre o braço superior do "F"
              Remete ao símbolo da seta/Finn
Cantos:       Rounded — 22% do tamanho (iOS padrão aplica automaticamente)
```

### 3.6 Fotografia e Imagens

**Estilo:** Pessoas reais, momentos cotidianos com dinheiro, leveza e realização — não stress.

**Incluir:**
- Pessoas olhando para o celular com expressão de satisfação/alívio
- Momentos de conquista: chaves de casa, viagem, celebração discreta
- Diversidade real brasileira: diferentes idades, etnias, contextos
- Luz natural, ambientes reais (não estúdio fake)

**Evitar:**
- Gráficos e planilhas em papel
- Pessoas com expressão de stress ou preocupação
- Imagens de banco (agência, cofre, notas de dinheiro empilhadas)
- Fotos com watermark ou aspecto stock genérico
- Pessoas brancas europeiziadas como único padrão

**Overlay em fotos com texto:** Navy-900 a 50-60% de opacidade.
