# Task: Conteúdo Emocional e Storytelling

**ID:** storyteller-conteudo-emocional
**Responsável:** storyteller
**Complexidade:** Média
**Duração Estimada:** 2 dias

## Descrição

Criar conteúdo emocional que complementa copywriting persuasivo. Inclui histórias de clientes, depoimentos estruturados, seções de impacto e narrativas de transformação.

## Entrada (Inputs)

- Narrativa da marca (storyteller-narrativa-marca)
- Copy principal (copywriter-copy-principal)
- Personas e comportamentos
- Dados de clientes (se disponível)

## Saída (Outputs)

- `emotional-content.md` - Conteúdo emocional estruturado
- `testimonial-framework.md` - Framework para depoimentos
- `customer-stories.json` - 3-5 histórias de clientes
- `impact-section.md` - Seção de impacto/transformação

## Checklist de Execução

- [ ] Identificar histórias de clientes reais
- [ ] Selecionar clientes que representam diferentes personas
- [ ] Estruturar depoimentos (problema, solução, resultado)
- [ ] Quantificar resultados onde possível
- [ ] Criar narrativa de transformação para cada história
- [ ] Desenvolver seção de impacto (lives changed, etc.)
- [ ] Criar mensagens de esperança e inspiração
- [ ] Desenvolver seção "Como começar" emocional
- [ ] Criar narrativas de "antes vs depois"
- [ ] Validar depoimentos com clientes
- [ ] Garantir permissão de uso de histórias
- [ ] Estruturar conteúdo em JSON para reutilização
- [ ] Criar versões de depoimentos por canal
- [ ] Revisar para autenticidade

## Critérios de Aceitação

- Mínimo 3-5 histórias de clientes completas
- Depoimentos autênticos e específicos
- Resultados quantificados onde apropriado
- Narrativas de transformação clara
- Conteúdo inspira e motiva
- Aprovado por clientes cujas histórias aparecem
- Pronto para integração no design
- Estrutura permite reutilização em múltiplos canais

## Dependências

- storyteller-narrativa-marca.md (MUST COMPLETE)

## Saída para

- storyteller-sequencia-comunicacao.md
- frontend-implementar-design.md (testimonials section)
- qa-analise-conversao.md (social proof impact)
