# Task: Relatório de Performance e Recomendações

**ID:** qa-relatorio-performance
**Responsável:** qa-analyst
**Complexidade:** Média
**Duração Estimada:** 2 dias

## Descrição

Compilar relatório completo de performance, testes e recomendações. Inclui análise consolidada, gargalos, oportunidades de melhoria e plano de ação.

## Entrada (Inputs)

- Testes de funcionalidade (qa-testes-funcionalidade)
- Análise de conversão (qa-analise-conversao)
- Performance audit (frontend-otimizar-performance)
- Todos os relatórios de fases anteriores

## Saída (Outputs)

- `FINAL-REPORT.md` - Relatório executivo
- `performance-metrics.json` - Métricas consolidadas
- `recommendations.md` - Plano de ação
- `qa-sign-off.md` - Documento de aprovação
- `deployment-checklist.md` - Pre-deployment checklist

## Checklist de Execução

- [ ] Compilar todos os resultados de testes
- [ ] Consolidar métricas de performance
- [ ] Análisar conversion rate vs benchmarks
- [ ] Identificar principais gargalos
- [ ] Quantificar impacto de bugs encontrados
- [ ] Avaliar qualidade geral (scoring)
- [ ] Documentar variação de performance
- [ ] Analisar experiência do usuário
- [ ] Criar recomendações priorizadas
- [ ] Estimar esforço de melhorias
- [ ] Identificar quick wins
- [ ] Documentar riscos conhecidos
- [ ] Preparar plano de monitoramento pós-launch
- [ ] Criar pre-deployment checklist
- [ ] Revisar com stakeholders

## Critérios de Aceitação

- Relatório abrangente e claro
- Todas as métricas importantes incluídas
- Recomendações priorizadas e realistas
- Riscos identificados e mitigados
- QA sign-off obtido
- Pre-deployment checklist completo
- Pronto para aprovação final
- Plano de monitoramento definido

## Dependências

- qa-testes-funcionalidade.md (MUST COMPLETE)
- qa-analise-conversao.md (MUST COMPLETE)
- frontend-otimizar-performance.md (RECOMMENDED)

## Saída para

- Aprovação final para deployment
- Planejamento de monitoramento pós-launch
