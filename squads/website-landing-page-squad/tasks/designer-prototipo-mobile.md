# Task: Design Prototype - Mobile

**ID:** designer-prototipo-mobile
**Responsável:** ux-designer
**Complexidade:** Média-Alta
**Duração Estimada:** 2-3 dias

## Descrição

Criar protótipo visual detalhado da landing page otimizado para dispositivos móveis. Inclui wireframes detalhados, componentes visuais, tipografia, cores e microinterações.

## Entrada (Inputs)

- Arquitetura da landing page (architect-estrutura-site)
- Mapa de conversão (architect-mapa-conversao)
- Estratégia (architect-estrategia-landing)
- Design system guidelines (se existente)
- Briefing visual do cliente

## Saída (Outputs)

- `prototype-mobile.figma` ou `.xd` - Arquivo de design interativo
- `mobile-design-spec.md` - Especificação de design detalhada
- `assets-mobile.zip` - Componentes e assets exportados
- `design-interactions.json` - Definição de microinterações

## Checklist de Execução

- [ ] Analisar arquitetura e wireframe base
- [ ] Definir breakpoints responsivos (320px, 375px, 425px)
- [ ] Criar grid e sistema de espaçamento para mobile
- [ ] Desenhar todas as seções da landing page
- [ ] Aplicar hierarquia visual apropriada para telas pequenas
- [ ] Definir tipografia legível em mobile
- [ ] Criar componentes reutilizáveis
- [ ] Adicionar estados interativos (hover, focus, active)
- [ ] Otimizar CTAs para toque (tamanho >= 44px)
- [ ] Exportar componentes em alta qualidade
- [ ] Revisar acessibilidade e contraste
- [ ] Validar com UX researcher

## Critérios de Aceitação

- Protótipo completo em ferramenta de design (Figma/XD/Sketch)
- Todas as seções desenhadas e otimizadas para mobile
- Componentes reutilizáveis documentados
- Especificação técnica incluindo margem, padding, tamanho de fonte
- CTAs destacados e touchable (mín. 44x44px)
- Microinterações documentadas
- Assets organizados e nomeados apropriadamente
