# Task: Mapa de Conversão e Fluxo de Usuário

**ID:** architect-mapa-conversao
**Responsável:** website-architect
**Complexidade:** Média
**Duração Estimada:** 1-2 dias

## Descrição

Detalhar o fluxo de conversão com pontos críticos de atrito, elementos de confiança, sequência de persuasão e call-to-actions estratégicos. Este documento guia design, copywriting e implementação.

## Entrada (Inputs)

- Estratégia de landing page (architect-estrategia-landing)
- Arquitetura da landing page (architect-estrutura-site)
- Personas de usuário
- Objetivos de conversão

## Saída (Outputs)

- `mapa-conversao-detalhado.md` - Documento completo de fluxo
- `conversion-funnel-visual.json` - Visualização do funil
- `trust-elements-map.json` - Mapa de elementos de confiança
- `cta-strategy.json` - Estratégia de Call-to-Action

## Checklist de Execução

- [ ] Mapear estágios do funil de conversão (awareness, interest, decision, action)
- [ ] Identificar pontos de atrito em cada estágio
- [ ] Definir elementos de confiança necessários (testimoniais, badges, garantias)
- [ ] Sequenciar elementos de persuasão (urgência, prova social, scarcity)
- [ ] Posicionar CTAs estrategicamente
- [ ] Definir microcopy persuasivo por seção
- [ ] Mapear comportamentos esperados do usuário
- [ ] Documentar triggers de conversão
- [ ] Criar visual do fluxo (diagrama ou JSON)
- [ ] Validar com team de marketing e design

## Critérios de Aceitação

- Funil mapeado com 4+ estágios claramente definidos
- Elementos de confiança especificados para cada seção
- CTAs posicionadas estrategicamente com copy definido
- Fluxo de usuário documentado e visualizado
- Pontos de atrito identificados e mitigações propostas
- Documentação compreensível para design e copywriting
