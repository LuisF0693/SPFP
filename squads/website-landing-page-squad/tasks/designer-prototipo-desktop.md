# Task: Design Prototype - Desktop

**ID:** designer-prototipo-desktop
**Responsável:** ux-designer
**Complexidade:** Média-Alta
**Duração Estimada:** 2-3 dias

## Descrição

Criar protótipo visual detalhado da landing page otimizado para desktop. Inclui wireframes detalhados, componentes visuais, tipografia, cores, efeitos visuais e microinterações.

## Entrada (Inputs)

- Arquitetura da landing page (architect-estrutura-site)
- Mapa de conversão (architect-mapa-conversao)
- Estratégia (architect-estrategia-landing)
- Design system guidelines (se existente)
- Briefing visual do cliente
- Brand guidelines (logo, cores, fonts)

## Saída (Outputs)

- `prototype-desktop.figma` ou `.xd` - Arquivo de design interativo
- `desktop-design-spec.md` - Especificação de design detalhada
- `assets-desktop.zip` - Componentes e assets em alta qualidade
- `design-animations.json` - Definição de animações e transições

## Checklist de Execução

- [ ] Analisar arquitetura e mapa de conversão
- [ ] Definir layout para desktop (1024px, 1280px, 1920px)
- [ ] Criar grid de 12 colunas e sistema de espaçamento
- [ ] Desenhar hero section com visual impactante
- [ ] Criar seções de features com cards visuais
- [ ] Desenhar social proof section (testimonials, logos)
- [ ] Criar pricing table (se aplicável)
- [ ] Desenhar footer com navegação e links
- [ ] Aplicar efeitos visuais (gradientes, sombras, ícones)
- [ ] Criar estados interativos (hover, focus, active)
- [ ] Definir animações de entrada (fade, slide, zoom)
- [ ] Adicionar guias de especificação técnica
- [ ] Exportar todos os componentes em múltiplas resoluções
- [ ] Revisar alinhamento e consistência visual
- [ ] Validar com arquiteto e copywriter

## Critérios de Aceitação

- Protótipo desktop completo em ferramenta de design
- Todas as seções desenhadas com qualidade profissional
- Componentes reutilizáveis com variações documentadas
- Especificação inclui todos os detalhes de spacing e tipografia
- Efeitos visuais claros e bem documentados
- Animações definidas e protótipadas
- Assets otimizados e em formato adequado (SVG/PNG/WebP)
- Revisado e aprovado por cliente/stakeholders
