# Task: Otimização On-Page e Conteúdo SEO

**ID:** seo-otimizacao-on-page
**Responsável:** seo-specialist
**Complexidade:** Média
**Duração Estimada:** 2 dias

## Descrição

Otimizar conteúdo da landing page para search engines. Inclui otimização de headlines, estrutura de headings, densidade de keywords, internal links e semântica HTML.

## Entrada (Inputs)

- Pesquisa de keywords (seo-keywords-research)
- Copy principal (copywriter-copy-principal)
- Arquitetura da landing page (architect-estrutura-site)
- Estratégia de landing page

## Saída (Outputs)

- `on-page-optimization.md` - Guia completo de otimizações
- `content-structure.json` - Estrutura otimizada de conteúdo
- `seo-checklist.md` - Checklist de implementação
- `semantic-markup.md` - Recomendações de markup semântico

## Checklist de Execução

- [ ] Revisar copy para densidade de keywords (1-2%)
- [ ] Validar estrutura de headings (H1 único, H2/H3 hierárquicos)
- [ ] Otimizar H1 com keyword principal
- [ ] Usar keywords em H2 onde natural
- [ ] Validar comprimento do meta description (150-160 chars)
- [ ] Revisar URL slug (descritivo, keywords se apropriado)
- [ ] Adicionar alt text a imagens (descriptivo + keyword quando natural)
- [ ] Identificar oportunidades de internal links
- [ ] Revisar estrutura de parágrafo (1-2 keywords por 100 words)
- [ ] Validar ligação semântica entre parágrafos
- [ ] Revisar para duplicação de conteúdo
- [ ] Otimizar schema markup (LocalBusiness, Product, etc.)
- [ ] Validar readability (Flesch reading score)
- [ ] Revisar interlinking com outras pages (se multi-page)

## Critérios de Aceitação

- H1 único com keyword principal
- Headings estruturados hierarquicamente
- Keywords naturalmente integradas
- Meta description otimizado
- Imagens com alt text descritivo
- Schema markup apropriado
- Conteúdo legível (Flesch score 50+)
- Sem keyword stuffing
- Internal links estrategicamente posicionados
