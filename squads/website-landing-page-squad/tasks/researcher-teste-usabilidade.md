# Task: Testes de Usabilidade

**ID:** researcher-teste-usabilidade
**Responsável:** ux-researcher
**Complexidade:** Média
**Duração Estimada:** 3 dias

## Descrição

Conduzir testes de usabilidade com protótipo ou landing page em desenvolvimento. Inclui testes de tarefa, navegação e análise de dificuldades encontradas.

## Entrada (Inputs)

- Protótipos (designer-prototipo-desktop, designer-prototipo-mobile)
- Landing page implementada (optional)
- Personas e objetivos

## Saída (Outputs)

- `usability-test-report.md` - Relatório detalhado
- `usability-metrics.json` - SUS, task completion, time-on-task
- `heatmaps.md` - Onde usuários clicam/scrollam
- `recommendations.md` - Recomendações de UX

## Checklist de Execução

- [ ] Recrutar 5-8 usuários para teste
- [ ] Preparar cenários de tarefa realistas
- [ ] Configurar ambiente de teste (lab, remote, etc.)
- [ ] Conduzir testes de usabilidade (30-45 min)
- [ ] Observar comportamento do usuário
- [ ] Registrar pontos de confusão
- [ ] Medir tempo para completar tarefas
- [ ] Coletar feedback pós-teste (SUS score)
- [ ] Análisar padrões de navegação
- [ ] Identificar pontos de clique mais comuns
- [ ] Detectar elementos ignorados
- [ ] Documentar frustrações verbalizadas
- [ ] Calcular task completion rate
- [ ] Compilar findings em relatório
- [ ] Priorizar recomendações por impacto

## Critérios de Aceitação

- Testes com mínimo 5 usuários
- Task completion rate > 80%
- SUS score >= 60
- Pain points identificados
- Recomendações actionable
- Antes/depois comparação possível
- Relatório com videos/screenshots
- Priorização de fixes clara

## Dependências

- designer-prototipo-desktop.md ou frontend-implementar-design.md

## Saída para

- researcher-analise-usuario-comportamento.md
- designer-prototipo-desktop.md (iteração)
- frontend-implementar-design.md (iteração)
