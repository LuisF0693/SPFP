# Task: Copy Principal e Headlines

**ID:** copywriter-copy-principal
**Responsável:** copywriter
**Complexidade:** Média
**Duração Estimada:** 2-3 dias

## Descrição

Escrever copy principal para todas as seções da landing page. Inclui headlines, subheadlines, descrições de features e mensagens persuasivas. Cada peça de copy é otimizada para conversão.

## Entrada (Inputs)

- Estratégia de landing page (architect-estrategia-landing)
- Mapa de conversão (architect-mapa-conversao)
- Personas de usuário
- Proposta de valor da empresa
- Protótipos de design (para entender layout)

## Saída (Outputs)

- `copy-principal.md` - Documento com toda copy principal
- `headlines-variations.json` - Variações de headlines para teste A/B
- `copy-specs.json` - Copy mapeada por seção/componente
- `messaging-framework.md` - Framework de mensagens-chave

## Checklist de Execução

- [ ] Analisar estratégia, personas e proposta de valor
- [ ] Revisar mapa de conversão para pontos-chave
- [ ] Escrever headline principal (compelling, clear, customer-focused)
- [ ] Escrever subheadline (suporta e expande na promise)
- [ ] Criar copy para hero section (call to action clara)
- [ ] Escrever descrição de cada feature (benefícios vs features)
- [ ] Criar copy de social proof section
- [ ] Escrever copy de seção de preços (se aplicável)
- [ ] Criar copy de objection handling
- [ ] Escrever copy de garantia/risco reversal
- [ ] Criar copy de footer e links
- [ ] Gerar 3-5 variações de headlines principais
- [ ] Validar tom de voz com brand guidelines
- [ ] Revisar para clareza e persuasão

## Critérios de Aceitação

- Headlines principais claros, compelentes e orientados para benefício
- Copy destaca benefícios (não apenas features)
- Linguagem apropriada para personas alvo
- Testes A/B headlines preparados
- Copy alinhado com ton de voz da marca
- Variações de copy para diferentes segmentos
- Nenhuma jargão desnecessário
- Validado com cliente ou stakeholder
