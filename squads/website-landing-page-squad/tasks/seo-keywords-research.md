# Task: Pesquisa de Keywords e Oportunidades SEO

**ID:** seo-keywords-research
**Responsável:** seo-specialist
**Complexidade:** Média
**Duração Estimada:** 2-3 dias

## Descrição

Pesquisar e validar keywords estratégicas para landing page. Inclui análise de volume de busca, intenção do usuário, competição e oportunidades de ranking.

## Entrada (Inputs)

- Estratégia de landing page (architect-estrategia-landing)
- Personas de usuário
- Objetivo de conversão
- Proposta de valor
- Análise de competidores (se disponível)

## Saída (Outputs)

- `keywords-research.md` - Relatório completo de pesquisa
- `keywords-target.json` - Keywords priorizadas com métricas
- `keyword-mapping.json` - Mapeamento de keywords por seção
- `seo-opportunities.md` - Oportunidades identificadas

## Checklist de Execução

- [ ] Identificar keywords seed relacionadas ao produto/serviço
- [ ] Pesquisar volume de buscas para cada keyword
- [ ] Analisar dificuldade de ranking (SEO difficulty)
- [ ] Verificar intenção de busca (commercial, transactional, etc.)
- [ ] Pesquisar long-tail keywords específicas
- [ ] Analisar top 10 resultados para cada keyword target
- [ ] Identificar gaps de conteúdo
- [ ] Avaliar autoridade dos competitors
- [ ] Priorizar keywords por volume + conversão + dificuldade
- [ ] Mapear keywords para seções da landing page
- [ ] Identificar quick wins (baixa dificuldade, volume)
- [ ] Pesquisar variações de busca (typos, sinônimos)
- [ ] Analisar featured snippets para keywords
- [ ] Criar lista final de 10-15 keywords target

## Critérios de Aceitação

- Mínimo 10-15 keywords target identificadas
- Cada keyword com volume estimado e dificuldade
- Keywords priorizadas por potencial de conversão
- Long-tail keywords inclusos
- Mapeamento claro de keywords para seções
- Análise de competidores incluída
- Oportunidades de quick wins identificadas
- Validado com arquiteto e copywriter
