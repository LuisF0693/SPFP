# Task: Pesquisa de Usuário

**ID:** researcher-pesquisa-usuario
**Responsável:** ux-researcher
**Complexidade:** Média
**Duração Estimada:** 2-3 dias

## Descrição

Conduzir pesquisa qualitativa com usuários alvo. Inclui entrevistas, surveys, análise de comportamento e validação de personas.

## Entrada (Inputs)

- Estratégia de landing page (architect-estrategia-landing)
- Personas definidas
- Objetivos de pesquisa

## Saída (Outputs)

- `user-research-report.md` - Relatório completo
- `personas-validated.json` - Personas validadas com dados
- `user-insights.md` - Insights principais
- `research-recordings.md` - Sumário de entrevistas

## Checklist de Execução

- [ ] Definir objetivos de pesquisa
- [ ] Recrutar 8-12 usuários de teste
- [ ] Preparar guia de entrevista
- [ ] Conduzir entrevistas qualitativas (30-45 min cada)
- [ ] Tomar notas e gravar (com consentimento)
- [ ] Análisar padrões de respostas
- [ ] Validar personas contra dados coletados
- [ ] Identificar pain points comuns
- [ ] Documentar motivações de usuários
- [ ] Analisar sentimentos e percepções
- [ ] Criar summary de insights
- [ ] Identificar oportunidades de melhoria
- [ ] Comparar com competidores mencionados
- [ ] Documentar surpresas/validações

## Critérios de Aceitação

- Entrevistas com mínimo 8 usuários
- Padrões identificados em respostas
- Personas validadas ou ajustadas
- Insights documentados
- Pain points mapeados
- Oportunidades identificadas
- Relatório apresentável
- Recomendações para design/copy

## Dependências

- architect-estrategia-landing.md (MUST COMPLETE)

## Saída para

- researcher-teste-usabilidade.md
- designer-prototipo-desktop.md (feedback)
- copywriter-copy-principal.md (feedback)
