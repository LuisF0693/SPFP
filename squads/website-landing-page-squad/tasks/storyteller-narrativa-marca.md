# Task: Narrativa da Marca

**ID:** storyteller-narrativa-marca
**Responsável:** storyteller
**Complexidade:** Média
**Duração Estimada:** 2 dias

## Descrição

Desenvolver narrativa de marca convincente e memorável. Inclui história da empresa, missão, valores e posicionamento emocional que ressoa com target audience.

## Entrada (Inputs)

- Briefing do cliente (negócio, história, valores)
- Personas de usuário (architect-estrategia-landing)
- Proposta de valor
- Insights de pesquisa de usuário (se disponível)

## Saída (Outputs)

- `brand-narrative.md` - Narrativa completa
- `brand-story.json` - Versões curta e longa
- `emotional-messaging.md` - Mensagens emocionais por persona
- `narrative-framework.md` - Framework de storytelling

## Checklist de Execução

- [ ] Entrevistar founder/CEO sobre história
- [ ] Documentar origem e motivação
- [ ] Identificar valor central e propósito
- [ ] Mapear problemas que empresa resolve
- [ ] Criar "before/after" narrative
- [ ] Develop 3-5 mini-stories de sucesso (clientes)
- [ ] Identificar emoções alvo (esperança, confiança, etc.)
- [ ] Criar versão curta (30 sec elevator pitch)
- [ ] Criar versão média (2-3 min)
- [ ] Criar versão longa (5+ min) com detalhe
- [ ] Validar autenticidade da narrativa
- [ ] Assegurar consistência com valores
- [ ] Criar messaging framework por persona
- [ ] Revisar com cliente para aprovação

## Critérios de Aceitação

- Narrativa autêntica e diferenciada
- Versões curta/média/longa criadas
- Apela emocionalmente a personas
- Propósito claro e inspirador
- Consistente com brand guidelines
- Pronto para usar em copy e design
- Aprovado pelo cliente
- Único e memorável

## Dependências

- architect-estrategia-landing.md (MUST COMPLETE)

## Saída para

- storyteller-conteudo-emocional.md
- copywriter-copy-principal.md (feedback)
- designer-prototipo-desktop.md (tone inspiration)
