## 1. BRIEFING DO PROJETO

### Objetivo Principal
**Múltiplo:** Lead + Venda + Consultoria

Estruturar um funil que captura leads, converte para plataforma (R$99,90/mês) e consultoria (R$349,90/mês).

### Público-Alvo
**Amplo:** Empreendedores, pessoas físicas, PMEs

### Problema Resolvido
⏰ **FALTA DE TEMPO** para gerenciar finanças

### Proposta Única
- Consultor IA integrado (automação)
- Interface intuitiva e fácil uso
- Preço mais competitivo
- Suporte e acompanhamento humano

### Preços
- **Plataforma:** R$99,90/mês (com Consultor IA)
- **Consultoria:** R$349,90/mês (consultoria humana + plataforma)
- **Modelo Híbrido:** Ambas as opções disponíveis

### Etapa 4: CONVERSION (Conversão)

**Objetivo:** Capturar lead ou fazer venda

**Touchpoint A: Form para Leads**

```
Campos:
1. Nome completo (obrigatório)
2. Email (obrigatório)
3. Telefone (opcional)
4. Tipo de interesse:
   ○ Apenas Plataforma
   ○ Apenas Consultoria
   ○ Ambos/Não tenho certeza
5. Desafio principal (dropdown):
   - Não sei onde vai meu dinheiro
   - Quero começar a investir
   - Preciso consolidar patrimônio
   - Tenho renda irregular
   - Outro

Confirmação:
- Email imediato: "Bem-vindo! Aqui está..."
- Acesso a: Webinar grátis / E-book / Demo
- SMS: Link de confirmação
```

**Touchpoint B: Checkout para Venda**

```
Integração: Stripe + Pix
Fluxo:
1. Escolher plano
2. Criar conta / Login
3. Dados de cobrança
4. Confirmar pagamento
5. Acesso imediato ao dashboard

Confirmação:
- Email: Boas-vindas + Acesso ao dashboard
- SMS: Código de acesso
- In-app: Tutorial de onboarding
```

**Métrica de Sucesso:**
- Conversion rate geral: 2-3%
- Cost Per Lead: < R$50
- Form abandonment: < 60%

### Etapa 5: RETENTION (Retenção/Expansão)

**Objetivo:** Transformar lead em cliente, cliente em expansion

**Email Sequence para Leads (Não converteu a venda)**

```
DIA 0: Confirmação
├─ Email: "Bem-vindo! Aqui está seu material"
├─ Conteúdo: E-book "5 Erros que Custam Dinheiro"
└─ CTA: "Agendar Consultoria" ou "Teste Grátis"

DIA 1-2: Educação
├─ Email: "3 dicas para começar a controlar finanças"
├─ Personalizado por persona
└─ CTA: "Ver Demo em Vídeo"

DIA 3-7: Valor
├─ Email: "Histórias de sucesso de usuários como você"
├─ Case study relevante
└─ CTA: "Comece Agora"

DIA 7: Decisão
├─ Email: "Sua análise está pronta (consultoria grátis)"
├─ Oferta especial: "Primeiros 7 dias = 50% off"
└─ CTA: Dupla (Plataforma ou Consultoria)

DIA 14: Re-engagement
├─ Email: "Último lembrete: Seu desconto de 50% expira"
└─ CTA: "Não perça a oferta"

DIA 21: Abandono
├─ Email: "Sentiremos sua falta. Que tal uma conversa?"
├─ Oferta: Chat com especialista grátis
└─ Status: Pausar sequência se não responder
```

**Como vencer:**

### Objeção 2: "Já tentei apps de finanças, não funciona"

**Onde aparece:** Hero + Features

**Como vencer:**
- Copy: "Diferente de apps abandonados"
- Feature: "Não está sozinho: IA + Consultoria humana"

### Objeção 3: "Por que pagar se Excel é grátis?"

**Onde aparece:** Pricing

**Como vencer:**
- Value math: "R$99,90 ÷ 30 = R$3,33/dia"

- Time: "Seu tempo vale mais que R$3,33"
- Proof: "Economia média dos usuários: R$2.340/mês"

### Objeção 4: "E se quiser cancelar?"

**Onde aparece:** FAQ + Footer

**Como vencer:**

- Proof: "Poucos cancelam (90% continuam)"

### Objeção 5: "Não tenho tempo de aprender"

- Headline: "Minutos, não horas"
