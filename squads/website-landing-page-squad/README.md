# Website Landing Page Squad

A comprehensive squad for creating high-performance landing pages through a multidisciplinary team of AI agents.

## Overview

The **Website Landing Page Squad** is designed to help entrepreneurs and small teams create professional, conversion-optimized landing pages without hiring external agencies. It brings together 9 specialized agents covering strategy, design, copywriting, SEO, development, and analysis.

## Workflows

### Landing Page Creation Workflow
Complete end-to-end process for creating a landing page:

1. **Strategy Phase** - Website Architect defines goals and structure
2. **Design Phase** - UX Designer creates visual prototypes
3. **Content Phase** - Copywriter and Storyteller craft messages
4. **SEO Phase** - SEO Specialist optimizes for search
5. **Development Phase** - Frontend/Backend Developers implement
6. **Testing Phase** - QA Analyst validates quality
7. **Optimization Phase** - QA Analyst optimizes for conversions

### Optimization Cycle
Continuous improvement workflow:

1. **Analysis** - Analyze current performance data
2. **Research** - UX Researcher gathers user feedback
3. **Hypothesis** - Architect creates optimization hypothesis
4. **Implementation** - Designers and developers implement changes
5. **Testing** - QA Analyst validates improvements
6. **Iteration** - Repeat based on results

## Best Practices

1. **Start with Strategy** - Always begin with the Website Architect
2. **User-Centered Design** - Validate with UX Researcher early
3. **Performance First** - Optimize for speed and Core Web Vitals
4. **Accessible Design** - Build with accessibility in mind
5. **Conversion Focused** - Every element should support conversion goal
6. **Test Continuously** - Regular testing and optimization cycles
7. **Mobile First** - Design and test mobile experience first
