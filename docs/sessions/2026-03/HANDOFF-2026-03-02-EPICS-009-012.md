## Padrões Arquiteturais desta Sessão

- **Padrão de widget especial em Squad**: `if (squad.name === 'X') return <ComponenteEspecial />;`
- **Validação HMAC em Edge Functions Deno**: usar `crypto.subtle` (Web Crypto API nativa)
- **Upsert com ignoreDuplicates** para webhooks idempotentes: `onConflict: 'user_id,source,external_id'`
- **Realtime subscription** em tabelas: `supabase.channel().on('postgres_changes', ...).subscribe()`
- **service_role policies** necessárias para Edge Functions acessarem tabelas com RLS
