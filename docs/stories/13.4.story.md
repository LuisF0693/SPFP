## User Story

**Como** desenvolvedor do SPFP
**Quero** um componente `FinnAvatar` reutilizável com os dois modos do Finn
**Para que** qualquer tela do app possa exibir o Finn com a expressão correta para o contexto

## Description

Criar o componente `FinnAvatar.tsx` usando os PNGs aprovados do Finn. Componente deve suportar dois modos (`advisor` e `partner`), quatro tamanhos e ser facilmente reutilizável em Insights, Onboarding, MonthlyRecap e qualquer outro componente que precise do Finn.

**Assets disponíveis:**
- `public/branding/finn-advisor.png` — Finn azul, expressão focada/analítica
- `public/branding/finn-partner.png` — Finn teal, expressão feliz/encorajadora

**Referência:** `squads/spfp-branding/outputs/BRAND-GUIDELINES-v1.1.md` seção 3.4

## Technical Notes

```tsx
// src/components/FinnAvatar.tsx
interface FinnAvatarProps {
  mode?: 'advisor' | 'partner';
  size?: 'sm' | 'md' | 'lg' | 'xl';
  className?: string;
  showLabel?: boolean;
}

const SIZES = { sm: 32, md: 40, lg: 80, xl: 120 };
const BORDERS = { advisor: '#1B85E3', partner: '#00C2A0' };
const IMAGES = {
  advisor: '/branding/finn-advisor.png',
  partner: '/branding/finn-partner.png',
};
const LABELS = { advisor: 'Finn · Advisor', partner: 'Finn · Parceiro' };
const ALTS = {
  advisor: 'Finn — modo analítico',
  partner: 'Finn — modo parceiro',
};

export const FinnAvatar: React.FC<FinnAvatarProps> = ({
  mode = 'advisor',
  size = 'md',
  className = '',
  showLabel = false,
}) => {
  const px = SIZES[size];
  return (
    <div className={`flex flex-col items-center gap-1 ${className}`}>
      <img
        src={IMAGES[mode]}
        alt={ALTS[mode]}
        width={px}
        height={px}
        style={{
          borderRadius: '50%',
          border: `2px solid ${BORDERS[mode]}`,
          objectFit: 'cover',
        }}
      />
      {showLabel && (
        <span style={{ fontSize: 11, color: BORDERS[mode], fontWeight: 600 }}>
          {LABELS[mode]}
        </span>
      )}
    </div>
  );
};
```
