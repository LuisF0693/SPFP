# Story 7.1 — Filtros Avançados no Relatório

## Status: Done

## Contexto
O módulo de Relatórios exibe dados fixos sem opção de filtrar por conta, categoria ou período customizado. Esta story adiciona painel de filtros avançados que atualiza os dados em tempo real.

## Acceptance Criteria

- [ ] AC1: Painel de filtros com: período (data início/fim), conta(s), categoria(s), tipo (receita/despesa/todos)
- [ ] AC2: Filtros aplicados atualizam todos os gráficos e tabelas do relatório em tempo real
- [ ] AC3: Botão "Limpar filtros" restaura para o estado padrão (mês atual, todas as contas)
- [ ] AC4: Filtros persistem durante a sessão (não resetam ao trocar de aba)
- [ ] AC5: Contador de filtros ativos visível no botão de filtros ("Filtros (3)")
- [ ] AC6: Seleção múltipla de contas e categorias
- [ ] AC7: Relatório sem dados exibe estado vazio informativo

## Arquivos a Modificar

- `src/components/Reports.tsx` — painel de filtros e lógica de filtragem
- Possível: novo componente `src/components/ui/ReportFilters.tsx`
