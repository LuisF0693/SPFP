# Story 11.1 — Content Workspace (Criar, Salvar e Organizar Conteúdos)

## Status: Done

## Contexto
O Marketing Hub começa com um workspace onde agentes e humanos criam, salvam e organizam conteúdos (posts, legendas, vídeos, imagens). É o "rascunho + biblioteca" de conteúdo antes da publicação.

## Acceptance Criteria

- [ ] AC1: Seção "Marketing Hub" acessível via Squad de Marketing no CRM Empresa
- [ ] AC2: Tipos de conteúdo: Post (texto + imagem), Reel/Vídeo, Story, Carrossel, Email
- [ ] AC3: Criar conteúdo: título, tipo, plataforma alvo, texto/legenda, mídia (imagem/vídeo)
- [ ] AC4: Upload de mídia para Supabase Storage (`content-media/{userId}/{contentId}/`)
- [ ] AC5: Preview do conteúdo antes de publicar (simula como vai aparecer)
- [ ] AC6: Status do conteúdo: Rascunho | Pronto para Publicar | Agendado | Publicado
- [ ] AC7: Agentes podem criar conteúdos via Supabase MCP (campo `created_by_agent`)

## Schema Supabase

```sql
CREATE TABLE marketing_content (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  title TEXT NOT NULL,
  content_type TEXT NOT NULL, -- post | reel | story | carousel | email
  platform TEXT[], -- instagram | facebook | youtube | email
  caption TEXT,
  hashtags TEXT[],
  media_urls TEXT[],
  status TEXT DEFAULT 'draft', -- draft | ready | scheduled | published
  scheduled_at TIMESTAMPTZ,
  published_at TIMESTAMPTZ,
  published_urls JSONB, -- {instagram: 'url', youtube: 'url'}
  created_by_agent TEXT, -- agent ID se criado por agente
  created_at TIMESTAMPTZ DEFAULT now()
);
```
