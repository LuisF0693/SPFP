# Story 11.3 — Meta Business API (Posts, Stories e Reels)

## Status: Done

## Contexto
Integrar o Marketing Hub com a Meta Business API para publicar conteúdo diretamente no Instagram e Facebook via SPFP, sem precisar sair da plataforma.

- [ ] AC3: Publicar Post no Facebook Page

- [ ] AC6: Tratamento de erros (token expirado, mídia inválida, rate limit)
- [ ] AC7: Log de publicações com status e erros

## Fluxo Técnico

```
Frontend SPFP
    ↓ (chama Edge Function com content_id)
Supabase Edge Function: publish-meta
    ↓ (busca conteúdo + credenciais no Supabase)
Meta Graph API:
  POST /me/media (upload de mídia)
    ↓ media_id
  POST /me/media_publish (publicar)
    ↓ post_id
Supabase: atualiza marketing_content.published_urls
```
