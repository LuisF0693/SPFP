# Story 8.2 — Contexto Financeiro Automático no Prompt

## Status: Done

## Contexto
O Gemini recebe prompts sem contexto financeiro do usuário, gerando respostas genéricas. Esta story injeta automaticamente um resumo financeiro do mês atual no system prompt, tornando as respostas muito mais precisas e personalizadas.

## Acceptance Criteria

- [ ] AC1: Resumo financeiro do mês atual injetado automaticamente no system prompt do Gemini
- [ ] AC2: Resumo inclui: receita total, despesa total, saldo líquido, top 3 categorias de gasto
- [ ] AC3: Resumo inclui: saldo de cada conta (cartões, corrente, investimentos)
- [ ] AC4: Resumo inclui: metas ativas com progresso % atual
- [ ] AC5: Dados usam transações reais do mês atual (não médias históricas)
- [ ] AC6: Badge visual "Com contexto financeiro ativo 🧠" visível no header do chat
- [ ] AC7: Usuário pode ver o contexto injetado (botão "Ver contexto")

## Arquivos a Modificar

- `src/components/Insights.tsx` — gerar e injetar resumo no prompt
- `src/services/aiService.ts` — system prompt dinâmico

## Tarefas

## Dev Notes
O resumo deve ser conciso (~300 tokens máx) para não consumir muito do contexto do Gemini. Formato sugerido:
```
CONTEXTO FINANCEIRO — [Mês/Ano]:
Receita: R$ X | Despesa: R$ Y | Saldo: R$ Z
Top gastos: Alimentação (R$X), Transporte (R$Y), Lazer (R$Z)
Contas: Nubank CC (-R$X), Itaú Corrente (R$Y)
Metas: Viagem (45%), Reserva (72%)
```
