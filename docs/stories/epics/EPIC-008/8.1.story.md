# Story 8.1 — Chips de Prompts Rápidos Sugeridos

## Status: Pending

## Contexto
O Consultor IA exige que o usuário saiba o que perguntar. Esta story adiciona chips de prompts rápidos acima do input, com perguntas frequentes e úteis pré-definidas.

## Acceptance Criteria

- [ ] AC1: Área de chips visível acima do campo de input quando não há mensagens
- [ ] AC2: Mínimo 6 chips de prompts sugeridos relevantes ao contexto financeiro
- [ ] AC3: Clicar em um chip preenche o input e envia automaticamente
- [ ] AC4: Chips rotativos — sorteio aleatório de um set maior a cada sessão
- [ ] AC5: Chips se ocultam após a primeira mensagem enviada (não poluem a conversa)
- [ ] AC6: Design visual: chips pill (rounded-full) com ícone + texto curto

## Chips Sugeridos (pool de 12+)

- "Onde posso economizar mais? 💡"
- "Qual minha maior despesa este mês? 🔍"
- "Estou no caminho certo com minhas metas? 🎯"
- "Compare meu gasto com o mês passado 📈"
- "Quais categorias estão acima do orçamento? ⚠️"
- "Dê um diagnóstico da minha saúde financeira 🏥"
- "Como está meu saldo em todas as contas? 💰"
- "Sugestões para aumentar minha reserva de emergência 🛡️"
- "Analise meu padrão de gastos este mês 🧠"
