# Story 9.2 — Squads: Espaços de Trabalho com Ícone e Cor

## Status: Done

## Contexto
Squads são os espaços de trabalho do CRM Empresa (equivalente a "Spaces" no ClickUp). Cada squad representa um time ou área (Marketing, Vendas, Ops, Admin, Produtos, CS). Esta story cria o CRUD de squads com identidade visual.

- [ ] AC2: Cada squad tem: nome, ícone (emoji), cor, descrição e membros
- [ ] AC3: Modal de criação/edição de squad com seletor de emoji e paleta de cores
- [ ] AC4: Squads listados na sidebar com ícone, cor e nome
- [ ] AC5: Clicar no squad abre a visão do squad (boards) na área principal
- [ ] AC6: Admin pode criar, editar e arquivar squads
- [ ] AC7: Dados persistem no Supabase (tabela `company_squads`)

## Schema Supabase

```sql
CREATE TABLE company_squads (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  name TEXT NOT NULL,
  icon TEXT DEFAULT '🏢',
  color TEXT DEFAULT '#6366f1',
  description TEXT,
  is_archived BOOLEAN DEFAULT false,
  sort_order INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);
-- RLS: user_id = auth.uid()
```
