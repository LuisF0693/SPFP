# Story 3.1 — Cartões Virtuais com Limite de Crédito Compartilhado

## Status: Pending

## Contexto
Usuários com cartões virtuais (ex: Nubank, C6, etc.) precisam que os cartões virtuais do mesmo cartão físico compartilhem um único limite de crédito. Atualmente cada cartão tem seu próprio limite independente.

## Acceptance Criteria

- [ ] AC1: Ao criar/editar cartão de crédito, opção "Cartão Virtual" com campo para vincular ao cartão físico pai
- [ ] AC2: Cartões virtuais vinculados ao mesmo cartão físico compartilham o limite total
- [ ] AC3: O limite utilizado é calculado somando todas as faturas dos cartões virtuais do mesmo grupo
- [ ] AC4: Na visualização de contas, cartões virtuais aparecem agrupados sob o cartão físico
- [ ] AC5: O saldo disponível mostrado considera todos os cartões do grupo
- [ ] AC6: Transações com cartão virtual afetam o limite do cartão físico pai
- [ ] AC7: Compatibilidade retroativa — cartões existentes sem vínculo continuam funcionando normalmente
