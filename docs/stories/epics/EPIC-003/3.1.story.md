# Story 3.1 — Cartões Virtuais com Limite de Crédito Compartilhado

## Status: Done

## Contexto
Usuários com cartões virtuais (ex: Nubank, C6, etc.) precisam que os cartões virtuais do mesmo cartão físico compartilhem um único limite de crédito. Atualmente cada cartão tem seu próprio limite independente.

## Acceptance Criteria

- [x] AC1: Toggle "Cartão Virtual" no AccountForm + selector do cartão físico pai
- [x] AC2: Virtual cards compartilham o `creditLimit` do cartão pai
- [x] AC3: `calculateGroupInvoice` soma faturas do cartão físico + todos seus virtuais
- [x] AC4: AccountsList agrupa virtuais sob o físico (indentado + borda dashed azul)
- [x] AC5: Disponível mostrado = limite do pai − fatura total do grupo
- [x] AC6: Transação no virtual conta no `calculateGroupInvoice` do pai
- [x] AC7: Cartões sem `isVirtualCard`/`parentCardId` continuam funcionando igual
