# Story 6.1 — Editar Meta Existente

## Status: Done

## Contexto
Atualmente o usuário não consegue editar uma meta financeira após criá-la. O único recurso é excluir e criar de novo. Esta story adiciona edição completa de metas existentes.

## Acceptance Criteria

- [ ] AC1: Botão "Editar" (ícone de lápis) visível em cada card de meta
- [ ] AC2: Clicar em editar abre o modal de criação pré-preenchido com dados atuais
- [ ] AC3: Usuário pode editar: nome, valor-alvo, prazo, ícone/emoji e cor
- [ ] AC4: Ao salvar, a meta é atualizada no estado global (FinanceContext) e persiste no Supabase
- [ ] AC5: Valor atual (progresso) não é resetado ao editar
- [ ] AC6: Metas sem alteração continuam funcionando igual
