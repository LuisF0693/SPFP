# 1.1: Modernizar UI/UX do CRM

**Status:** Draft
**Sprint:** 2
**Points:** 8
**Epic:** EPIC-001

---

## User Story

**Como** planejador financeiro
**Quero** uma interface moderna e limpa do CRM
**Para que** eu tenha uma experiência profissional e competitiva

---

## Description

Redesenhar a interface do AdminCRM com design system consistente, usando glassmorphism, animações suaves e layout responsivo. Melhorar visualização de clientes com cards expandíveis, health score visual (gauge), grid com filtros avançados e busca com autocomplete.

---

## Acceptance Criteria

### AC-001.1: Design System Consistente
- [ ] Paleta de cores definida e documentada
- [ ] Componentes reutilizáveis (buttons, cards, inputs)
- [ ] Glassmorphism aplicado em cards principais
- [ ] Tipografia consistente em toda interface

### AC-001.2: Cards de Cliente Melhorados
- [ ] Cards mostram: avatar, nome, email, telefone, health score
- [ ] Cards expandíveis com ações adicionais
- [ ] Hover effect com shadow/gloss
- [ ] Status visual (ativo, inativo, pendente)

### AC-001.3: Animações e Transições Suaves
- [ ] Uso de framer-motion para transições
- [ ] Fade in/out em carregamento
- [ ] Slide de painéis laterais
- [ ] Bounce em cards ao hover

### AC-001.4: Dark Mode Otimizado
- [ ] Cores com bom contraste (AA WCAG)
- [ ] Backgrounds: #0f1419, cards: #1a1f2e
- [ ] Sem cores muito saturadas
- [ ] Toggle dark/light funcional

### AC-001.5: Layout Responsivo
- [ ] Desktop: 3 colunas de cards
- [ ] Tablet: 2 colunas
- [ ] Mobile: 1 coluna, cards full width
- [ ] Sem overflow horizontal

### AC-001.6: Skeleton Loading
- [ ] Skeleton cards durante carregamento
- [ ] Animação shimmer
- [ ] Deve completar em < 2 segundos

### AC-001.7: Empty States
- [ ] Ilustrações para estado vazio
- [ ] Mensagens claras (ex: "Nenhum cliente ainda")
- [ ] CTA (ex: "Criar Cliente")
