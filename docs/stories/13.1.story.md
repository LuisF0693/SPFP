## User Story

**Como** usuário do SPFP
**Quero** ver a interface com as cores e tipografia corretas da nova identidade visual
**Para que** a experiência reflita a marca premium do SPFP desde a primeira interação

---

## Description

Atualizar o `tailwind.config.js` com a nova paleta de cores aprovada nas Brand Guidelines v1.1 e substituir a fonte Inter pela Plus Jakarta Sans no `index.html`. Esta story é **fundação** — todas as outras stories do EPIC-013 dependem dela.

**Referência:** `squads/spfp-branding/outputs/BRAND-GUIDELINES-v1.1.md` seção 3.2

## Acceptance Criteria

### AC-1301.1: Paleta de cores atualizada no Tailwind
- [ ] `tailwind.config.js` contém as cores: `navy-900: '#0A1628'`, `navy-700: '#1B3A6B'`, `blue-logo: '#1B85E3'`, `blue-300: '#6AA9F4'`, `blue-100: '#E8F1FD'`
- [ ] Cores teal adicionadas: `teal-500: '#00C2A0'`, `teal-200: '#99EAD8'`, `teal-100: '#CCFAF3'`
- [ ] Cores de feedback mantidas: `success: '#10B981'`, `error: '#EF4444'`, `warning: '#F59E0B'`, `info: '#3B82F6'`
- [ ] Cores antigas (`primary: #000000`, `accent: #3b82f6` genérico) não removidas — apenas adicionadas as novas para não quebrar componentes existentes

### AC-1301.2: Plus Jakarta Sans importada e configurada
- [ ] Google Fonts import no `index.html` para Plus Jakarta Sans (weights: 400, 500, 600, 700, 800)
- [ ] `tailwind.config.js` `fontFamily.sans` inclui `'Plus Jakarta Sans'` como primeira opção
- [ ] Sora importada para uso em headlines de display (weight: 800)
- [ ] Inter mantida como fallback (não remover — evita regressão)

### AC-1301.3: Sem regressão visual
- [ ] App carrega sem erros de console
- [ ] Build (`npm run build`) passa sem erros
- [ ] Componentes existentes não quebram visualmente

## Technical Notes

```js
// tailwind.config.js — adicionar em theme.extend.colors:
colors: {
  'navy-900': '#0A1628',
  'navy-700': '#1B3A6B',
  'blue-logo': '#1B85E3',
  'blue-300': '#6AA9F4',
  'blue-100': '#E8F1FD',
  'teal-500': '#00C2A0',
  'teal-200': '#99EAD8',
  'teal-100': '#CCFAF3',
}

// tailwind.config.js — fontFamily.sans:
fontFamily: {
  sans: ['Plus Jakarta Sans', 'Inter', 'system-ui', 'sans-serif'],
  display: ['Sora', 'Plus Jakarta Sans', 'sans-serif'],
}
```

```html
<!-- index.html — Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=Sora:wght@800&display=swap" rel="stylesheet">
```
