## Acceptance Criteria

### AC-1308.1: Identidade visual antiga removida
- [ ] Nenhum spirograph ou elemento SVG do logo antigo visível em nenhuma tela
- [ ] Nenhuma referência ao "LF" no layout do app (exceto dado de usuário)
- [ ] Favicon atualizado (verificar na aba do browser)
- [ ] Meta title correto: "SPFP — Seu Planejador Financeiro Pessoal"

### AC-1308.2: Paleta de cores aplicada
- [ ] Background escuro principal é navy-900 (#0A1628) — não preto puro (#000)
- [ ] Acentos azuis usam blue-logo (#1B85E3) — não #3b82f6 genérico nas áreas rebrandadas
- [ ] Finn Advisor usa borda azul (#1B85E3)
- [ ] Finn Partner usa borda teal (#00C2A0)
- [ ] Nenhuma cor fora da paleta aprovada nas áreas novas

### AC-1308.3: Tipografia
- [ ] Plus Jakarta Sans carregando (verificar Network tab no DevTools)
- [ ] Títulos principais usam Plus Jakarta Sans Bold/ExtraBold
- [ ] Sem mistura com Playfair Display nas telas rebrandadas

### AC-1308.4: Finn consistente
- [ ] "Finn" nomeado corretamente em Insights, Onboarding e MonthlyRecap
- [ ] FinnAvatar exibe corretamente nos 3 locais (sem imagem quebrada)
- [ ] Modo advisor em análises, modo partner em celebrações

### AC-1308.5: Onboarding funcional
- [ ] Onboarding exibe para usuário novo (sem transações, sem flag)
- [ ] Onboarding NÃO exibe para usuário com histórico
- [ ] Todas as 5 telas navegam corretamente
- [ ] Botão "Pular" funciona
- [ ] Botão "Registrar agora" na tela 5 abre formulário de transação
- [ ] Após completar: onboarding não abre no próximo login

### AC-1308.6: Sales Page
- [ ] Headline: "A consultoria do Luis. A inteligência do Finn."
- [ ] Tagline: "Consultoria especializada. Inteligência que nunca dorme."
- [ ] Badge: "Planejamento Financeiro Pessoal"
- [ ] Visual navy-900 aplicado

### AC-1308.7: WCAG AA mantido
- [ ] Texto branco sobre navy-700 ≥ 4.5:1 ✅
- [ ] Texto branco sobre blue-logo — usar apenas para textos grandes (>18px)
- [ ] Finn messages legíveis nos dois modos
- [ ] Nenhum texto novo abaixo do contraste mínimo

### AC-1308.8: Sem regressão funcional
- [ ] Login/logout funciona
- [ ] Adicionar transação funciona
- [ ] Dashboard carrega dados corretamente
- [ ] Insights/Finn responde perguntas
- [ ] Build sem erros: `npm run build`
- [ ] Sem erros de console em nenhuma rota
