# 13.5: Insights — Finn com Identidade Visual e Brand Voice

**Status:** Ready
**Sprint:** 14
**Points:** 5

## User Story

**Como** usuário do SPFP
**Quero** interagir com o assistente de IA que se chama Finn, tem um rosto e fala de forma humana
**Para que** a experiência de planejamento financeiro seja pessoal, não genérica

---

## Description

Atualizar `Insights.tsx` para usar o nome "Finn" em toda a interface, integrar o componente `FinnAvatar` no header do chat (modo advisor por padrão), e aplicar o brand voice aprovado nos textos da interface. Substituir referências genéricas ("Bot", "IA", "Assistente") por "Finn".

**Referência:** `squads/spfp-branding/outputs/BRAND-VOICE-v1.1.md` seções 2 e 7

---

## Acceptance Criteria

### AC-1305.1: Nome "Finn" aplicado
- [ ] Header do chat exibe: `FinnAvatar (mode="advisor", size="md")` + texto `"Finn"`
- [ ] Subtítulo do header: `"Seu assistente financeiro pessoal"`
- [ ] Nenhuma referência a "Bot", "IA Financeira", "Assistente Virtual" visível na interface
- [ ] Mensagens do sistema enviadas pelo Finn identificadas com o nome "Finn" (não "Assistente")

### AC-1305.2: Placeholder e empty state
- [ ] Input placeholder: `"Pergunte ao Finn sobre suas finanças..."`
- [ ] Estado vazio (sem histórico): exibe `FinnAvatar (size="lg", mode="advisor")` centralizado
- [ ] Mensagem empty state: `"Oi! Eu sou o Finn. Pergunte sobre seus gastos, metas ou investimentos."`

### AC-1305.3: Brand voice nas mensagens de sistema
- [ ] Mensagem de boas-vindas (primeira vez): usa copy do Brand Voice — tom humano, sem jargão
- [ ] Mensagem de erro de IA: `"Algo deu errado aqui. Tenta de novo em instantes."` (não "Erro ao processar requisição")
- [ ] Mensagem de loading: `"Finn está analisando..."` (não "Carregando...")

### AC-1305.4: FinnAvatar no modo parceiro em celebrações
- [ ] Quando o Finn identifica uma meta atingida na resposta, exibir `FinnAvatar (mode="partner")`
- [ ] Implementação simples: se a resposta contiver palavras como "parabéns", "meta atingida", "concluíd", usar modo partner — caso contrário, advisor

### AC-1305.5: Paleta aplicada na UI do chat
- [ ] Bolhas de mensagem do Finn: fundo `blue-100` (#E8F1FD), texto `gray-900`
- [ ] Bolhas do usuário: fundo `navy-700` (#1B3A6B), texto branco
- [ ] Header do chat: fundo `navy-900` (#0A1628)

## Technical Notes

**Palavras para detectar modo partner:**
```ts
const PARTNER_KEYWORDS = ['parabéns', 'meta atingida', 'concluíd', 'conseguiu', 'alcançou', 'excelente'];
const isPartnerMode = (text: string) =>
  PARTNER_KEYWORDS.some(k => text.toLowerCase().includes(k));
```
