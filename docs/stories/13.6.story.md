## User Story

**Como** novo usuário do SPFP
**Quero** ser guiado por 5 telas de boas-vindas que me apresentam o produto e o Finn
**Para que** eu entenda o valor do SPFP antes de começar a usar e me sinta confortável na plataforma

---

## Description

Criar o componente `Onboarding.tsx` com 5 telas em carrossel. Exibir para novos usuários no primeiro acesso (verificar flag `onboardingCompleted` no perfil). O onboarding deve ter um visual premium alinhado ao brand (navy, plus jakarta sans, finn avatar). A tela 3 é o momento de apresentação do Finn.

**Copy aprovado:** `squads/spfp-branding/outputs/BRAND-VOICE-v1.1.md` seção 5

## Technical Notes

**Trigger no App.tsx:**
```tsx
// Após autenticação, antes de renderizar o layout:
const { onboardingCompleted, transactions } = useFinance();
const isNewUser = !onboardingCompleted && transactions.length === 0;

// Renderizar:
{isNewUser && <Onboarding onComplete={handleOnboardingComplete} />}
```

**Flag no FinanceContext (ou localStorage):**
```ts
// localStorage fallback simples:
const ONBOARDING_KEY = `spfp_onboarding_${user?.id}`;
const onboardingCompleted = localStorage.getItem(ONBOARDING_KEY) === 'true';
const completeOnboarding = () => localStorage.setItem(ONBOARDING_KEY, 'true');
```
