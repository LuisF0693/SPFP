## Technical Notes

```tsx
// src/components/Logo.tsx — nova implementação
interface LogoProps {
  variant?: 'full' | 'icon';
  size?: number;
  className?: string;
}

export const Logo: React.FC<LogoProps> = ({ variant = 'full', size, className }) => {
  if (variant === 'icon') {
    return (
      <img
        src="/branding/app-icon.png"
        alt="SPFP"
        width={size ?? 40}
        height={size ?? 40}
        className={className}
        style={{ objectFit: 'contain' }}
      />
    );
  }
  return (
    <img
      src="/branding/logo.png"
      alt="SPFP Logo"
      width={size ?? 160}
      className={className}
      style={{ objectFit: 'contain' }}
    />
  );
};
```

**Atenção:** O logo tem fundo transparente — vai se adaptar tanto a fundos claros quanto escuros. Testar em ambos os modos.
